## 7. 总结

### 缺失功能（Demo 有，实现缺失或禁用）

1. **账号行拖拽排序** — 设置页多账号组内的拖拽 grip 排序
2. **拖拽两步确认** — 主面板卡片拖拽的 armed 预确认机制

### 主要偏差（Demo 与实现不同）

1. **进度条形态** — Demo 是窄条+独立百分比列，实现是宽条+内嵌百分比
2. **周期颜色** — Demo 蓝色/紫色区分 5 小时/一周，实现统一蓝色
3. **状态文案** — Demo 有多种具体状态描述，实现只有 "运行中"/"刷新异常"
4. **卡片间距** — Demo 用 flex gap，实现额外加了 margin-bottom 导致间距翻倍
5. **设置窗口布局** — Demo 独立弹出窗口带圆角，实现是全屏无圆角窗口
6. **L2 切换文案** — Demo "平均" vs 实现 "概览"
7. **托盘菜单** — Demo 是自定义 React 渲染的圆角毛玻璃面板，实现是系统原生菜单，视觉完全不同
8. **托盘左键配置** — Demo 支持"打开主面板"或"打开菜单"切换，实现固定打开主面板
9. **主面板深色模式** — Demo 有完整 dark token 且主面板正常切换，实现主面板不响应深色主题切换

### 额外功能（实现有，Demo 无）

1. **导入/导出设置** — 导出 JSON 配置和导入 JSON 配置
2. **offscreen 高度镜像 DOM** — 用于精确控制弹出窗口高度
3. **动态插件参数表单** — `SettingsForm` 根据 plugin metadata 动态渲染字段
4. **CPA Connector 设置组件** — `CpaConnectorSettings` 独立组件
5. **更多 vendor 支持** — 支持 Gemini/Antigravity/Kimi（Demo 仅 6 个）
6. **macOS 标题栏拖拽适配** — `titlebar-no-drag` 类名
