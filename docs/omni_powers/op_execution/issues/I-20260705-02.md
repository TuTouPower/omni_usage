## 描述

`endpointOverrides` 可由用户配置/导入。`apply_auth` 按 manifest 把 secret 注入到解析后的端点（bearer/header/query）。

**风险**：导入恶意配置把连接器端点改指**公网攻击者主机**，现存 vault secret 会被发过去。`assert_safe_connector_host` 只拦云元数据主机（`169.254.169.254` / `metadata.google.internal` / `metadata.azure.com`），**不拦公网主机**（无法区分合法外部 API 与攻击者主机）。

**待办（需评估）**：导入后被改动端点的连接器要求**重新录入 secret**；或导入时对端点变更显式确认。

**相关 commit**：`ab96616 fix(security): block connector requests to cloud-metadata hosts`（已拦云元数据，公网仍开）。

**blueprint 现状记录**：`op_blueprint/specs/secret-vault.md` 已知限制段 + `connector-runtime.md` NetClient SSRF 段。
