# 用量条颜色方案设计规范

_来源：chat28 + chat29（2026-06-05 ~ 06）_

---

## 一、三套颜色方案

可在 `设置 > 外观 > 用量条颜色方案` 切换，排序固定：

| 序号 | 方案                           | 说明                                           |
| ---- | ------------------------------ | ---------------------------------------------- |
| 1    | **风险色：仅当前用量**（默认） | 只看 `current`，不依赖重置时间                 |
| 2    | 风险色：带投影预测             | 看 `current` + `projected`，不能算时回退方案 1 |
| 3    | 彩色区分：九色循环             | 纯视觉区分，不表达风险                         |

---

## 二、风险色：仅当前用量（默认）

只看当前用量百分比 `current`，不依赖任何时间数据。

```
current >= 95%      红色
current >  85%      橙色
current >  60%      黄色
其他                绿色
```

---

## 三、风险色：带投影预测

在当前用量基础上，按已过时间预测窗口结束时的累计用量。

```
projected = current / elapsed
```

- `current`：当前用量比例（0..1）
- `elapsed`：已过时间比例（0..1）

```
current >= 95% 或 projected >= 1.00    红色
current >  85% 或 projected >= 0.90    橙色
current >  60% 或 projected >= 0.75    黄色
其他                                   绿色
```

**回退条件**：`elapsed` 缺失、≤0、无重置时间、无窗口起点时，自动回退到「仅当前用量」规则。

---

## 四、彩色区分：九色循环

不表达风险，只用于区分不同用量条。按条目顺序循环取色，第 10 条起复用。

色板（3 + 3 + 3 冷色系）：

```
第 1 组（主色）
  #5B8CFF  主蓝
  #8B72F8  主紫
  #46C7C7  主青

第 2 组（次色）
  #7EA2FF  扩展蓝
  #A18CFF  扩展紫
  #72D4D1  扩展青

第 3 组（弱色）
  #9CB8FF  浅蓝灰
  #B6A7FF  浅紫灰
  #A7D8D8  淡青灰
```

同一卡片内严格按顺序分配，第 10 条 = `#5B8CFF`，第 11 条 = `#8B72F8`，依此类推。

---

## 五、通用规则

- **轨道底色**：`#E9EDF5`，所有方案、所有卡片统一，不随厂商或额度类型变化
- **默认方案**：必须是「风险色：仅当前用量」，禁止默认启用九色循环
- **风险色**：只用绿/黄/橙/红四色，亮暗主题各一套（通过 CSS 变量 `--risk-green` 等）
- **九色循环**：按位置索引 `index % 9` 取色，不按厂商或指标类型取色
- **无 elapsed 数据时**：投影方案自动降级为仅当前用量方案
